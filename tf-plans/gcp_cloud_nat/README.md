# GCP Cloud NAT Deployment Plan

Ready-to-use Terraform wrapper for deploying one or many Cloud NAT configurations using the reusable module at [modules/networking/gcp_cloud_nat](../../modules/networking/gcp_cloud_nat/README.md).

> Part of [gcp.tf-modules](../../README.md)

---

## Architecture

```text
tf-plans/gcp_cloud_nat/
└── module "cloud_nat"  →  modules/networking/gcp_cloud_nat/
    ├── google_compute_router.router      (optional)
    ├── google_compute_router_nat.nat_standard
    └── google_compute_router_nat.nat_dynamic
```

---

## Prerequisites

- Terraform `>= 1.5`
- Google provider `>= 6.0`
- Authenticated `gcloud` / Application Default Credentials
- Required APIs: `compute.googleapis.com`
- IAM permissions for router/NAT creation in target projects

---

## Files

| File | Purpose |
|------|---------|
| [main.tf](main.tf) | Calls the reusable Cloud NAT module |
| [variables.tf](variables.tf) | Wrapper input variables |
| [locals.tf](locals.tf) | `created_date` metadata for governance tags |
| [outputs.tf](outputs.tf) | Pass-through outputs from the module |
| [providers.tf](providers.tf) | Provider and optional backend configuration |
| [terraform.tfvars](terraform.tfvars) | Example values for one or many NATs |

---

## Required Variables

| Variable | Type | Description |
|----------|------|-------------|
| `project_id` | `string` | Default project ID for NAT definitions. |
| `nats` | `list(object)` | One or many NAT definitions. |

---

## Optional Variables

| Variable | Type | Default | Description |
|----------|------|---------|-------------|
| `region` | `string` | `"us-central1"` | Default region for NAT definitions. |
| `tags` | `map(string)` | `{}` | Common governance tags merged with `created_date` and `managed_by`. |

---

## Quick Start

```bash
cd tf-plans/gcp_cloud_nat
terraform init
terraform validate
terraform plan
terraform apply
```

---

## Outputs

| Output | Description |
|--------|-------------|
| `nat_ids` | NAT IDs keyed by NAT key |
| `nat_names` | NAT names keyed by NAT key |
| `nat_router_names` | Effective router names used by NATs |
| `created_router_names` | Routers created by this deployment |
| `nat_regions` | Resolved regions per NAT key |
| `nat_projects` | Resolved projects per NAT key |
| `common_tags` | Governance metadata generated by locals |
